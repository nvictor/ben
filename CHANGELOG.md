# BEN Changelog

All notable changes to the Bit Engineering Notation (BEN) specification will be documented in this file.

## [1.0.0] - 2026-02-16

### Added
- Initial formalized specification.
- Core axes: Form (F), Mechanism (M), and Status (S).
- Modifiers: Callback (C), Topper (T), and Act-out (A).
- Structural decomposition of comedy bits.
